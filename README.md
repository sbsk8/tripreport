# たびレポート

php自作課題

## 概要

旅行先の記録ができるサービスです。

公開･非公開の処理ができるようになっており、自分だけの記録としてもお使いになれます。



## 使い方

会員登録したユーザーのみがお使いになれます。

旅行先の記録の登録･編集･削除・公開された投稿にお気に入りが行えます。

お気に入り機能がついているので気になった他の人の公開された旅の記録はまとめて閲覧が可能です。</br>
